## 🚀 How to Use

### Prerequisites
```
✓ Apache/LAMPP installed
✓ Python 3.x installed
✓ Project location: /opt/lampp/htdocs/SIEMproject/
```

### 3-Step Startup

```bash
# Step 1: Start Apache
sudo /opt/lampp/lampp start

# Step 2: Start Python Script
cd /opt/lampp/htdocs/SIEMproject
python3 pythonSIEMscript.py

# Step 3: Open Website
# Visit: http://localhost/SIEMproject/
```

### Access Points

| URL | Purpose |
|-----|---------|
| `http://localhost/SIEMproject/` | Dashboard (main) |
| `http://localhost/SIEMproject/?action=logs` | View logs |
| `http://localhost/SIEMproject/?action=sync_status` | Monitor sync |

## 🐛 Quick Troubleshooting

| Issue | Solution |
|-------|----------|
| Apache not running | `sudo /opt/lampp/lampp start` |
| API not responding | Check Apache logs, restart Apache |
| Events not appearing | Verify `PHP_API_ENABLED = True` |
| Permission denied | `chmod 755 captured_logs/ && chmod 666 *.json` |
| Python script won't start | Check Python 3.x installed, no port conflicts |
